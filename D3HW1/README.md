# D3 Homework 1

This project uses D3.js to create a scatterplot visualization.

The scatterplot shows the relationship between:
- Commute Time (Minutes)
- Happiness Score

The chart demonstrates how longer commute times are associated with lower happiness scores.

The visualization loads data from an external CSV file and uses D3 scales, axes, and circles to display the data points.

## Files Included

- index.html
- main.js
- style.css
- data.csv
- screenshot.png

## Data Source

Sample dataset created for educational purposes.
