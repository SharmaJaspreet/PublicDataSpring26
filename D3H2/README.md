# D3 Multi-Line Chart

This visualization shows changes in bench press and squat strength over five months using D3.js.

Features:
- external CSV data
- SVG
- D3 line generator
- CSS styling
- SVG text labels
- multi-line chart

Data Source:
Self-created workout dataset for assignment practice.
